TBOSDT Command Reference

 
This following sections divide the TBOSDT command set into 6 functional groups:
 
  • Commands for Working with Files and Folders.
  • Commands for Working with Drives and File Systems.
  • Commands for Working with Registry Files (Hive Files).
  • Commands for Working with the Registry on a Running Windows System.
  • Commands Associated with Windows Driver Installation.
  • Additional Commands.
 
Each section will include a table summarizing the syntax and options for each command in the group, as well as some notes and examples of usage. While the majority of the commands are available in all 3 OS versions, there are some that are only available in the Windows version.
These will be noted in the command reference tables.
